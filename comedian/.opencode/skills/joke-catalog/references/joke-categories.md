# Joke Categories

## Puns (category: "pun")
Wordplay exploiting multiple meanings. Example: "I'm reading about anti-gravity. Impossible to put down!"

## One-Liners (category: "one-liner")
Single sentence, immediate payoff. Example: "I used to be indecisive, now I'm not so sure."

## Knock-Knock (category: "knock-knock")
Format: Knock knock / Who's there? / [Setup] / [Setup] who? / [Punchline]

## Dad Jokes (category: "dad-joke")
Wholesome, corny, groan-worthy. Example: "I'm afraid for the calendar. Its days are numbered."
